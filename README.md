##  Lab Objectives
This lab demonstrates common attack vectors against MySQL databases, including:
- SQL injection techniques
- Authentication bypass methods
- Database enumeration
- Data extraction vulnerabilities
- Security best practices for database hardening

---

##  Tools & Technologies Used
- MySQL Database Server
- Penetration testing tools
- Network analysis tools
- SQL injection frameworks

---

##  Security Recommendations
- Implement parameterized queries to prevent SQL injection
- Use strong authentication mechanisms
- Apply principle of least privilege for database users
- Enable MySQL audit logging
- Keep MySQL server updated with latest security patches
- Use firewalls to restrict database access
